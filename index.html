Goals of this project

I used to be a pretty decent self-taught web developer, but haven't built any new projects recently. This project will be a web app that allows a user to log in and add microgoals, defined as tasks that take 10 minutes or less, that users can add for four different areas of their lives: health, self, work, and relationships. The user will be able to add microgoals and mark tasks as completed.

My main goals for buiding this app include:

  1. Refresh my knowledge on HTML and CSS
  2. Build a couple mini apps that will help me build this larger app
  3. Build a full-stack app
